## About ChaBok

ChaBok is a real-time chat room application built on top of AJAX, providing fast, lightweight, and responsive messaging without requiring page reloads.  
The system is developed and maintained by **Saeed Shanaqi**.

ChaBok aims to deliver a smooth and seamless communication experience using modern web technologies while keeping the architecture clean, scalable, and easy to extend.

### Key Features

- **Real-time Messaging**  
  Instant chat updates without page refresh using efficient AJAX requests.

- **Lightweight Architecture**  
  Optimized for performance with minimal server load.

- **User-friendly Interface**  
  Simple and intuitive UI designed for fast communication.

- **Easy to Deploy**  
  Straightforward installation process suitable for developers of all skill levels.

- **Modular & Extensible**  
  Built with a structure that allows developers to add new features easily.

## Installation

Follow the steps below to install and set up the ChaBok application:

### 1. Extract the Project Files  
Unzip the downloaded package into your desired directory.

### 2. Install Composer Dependencies  
Navigate into the project directory and run:

```bash
composer install
```

### 3. Create the .env File
Copy the example environment file:

```bash
cp .env.example .env
```

### 4. Start the Installer
After completing the above steps, open the installation URL in your browser:

```bash
https://example.com/install
```

Follow the on-screen instructions to complete the setup.
